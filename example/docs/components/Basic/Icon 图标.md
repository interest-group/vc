# Icon 图标

提供了一套常用的图标集合。

## 基本使用

直接通过设置名字为 `name` 来使用即可。例如：

::: demo 
``` html
<v-icon name="home"></v-icon> 
<v-icon name="tv"></v-icon> 
<v-icon name="laptop"></v-icon>
```
:::

## API

### Icon props

| 属性     | 说明          | 类型    | 默认值  |
| -------- | ------------ | ------- | ------- |
| name     | 图标名称 | String  |        |
| icon     | 自定义图标     | String  |      |
