## API

### props

| 属性     | 说明          | 类型    | 默认值  |
| -------- | ------------ | ------- | ------- |
| target     | 滚动节点，留空则自动查找可滚动父级节点 | HTMLElement  |        |
| offset     | 滚动显示距离     | Number  |  0    |
