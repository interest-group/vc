
```

```

```

```

    ├── components // 组件

### 关于组件文件

格式：

    ./src/components/button/
    ├── Button.vue
    └── index.js 必须

`index.js`
```js
// 引入并导出组件
import Button from './Button'

// ...

export default Button
```

按照这种格式创建组件，不需要再其他地方引入注册，可以直接在案例代码中使用，如：

```html
<vc-button>..</vc-button>
```
